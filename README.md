# myInvoice GitHub Pages Site

Static website for myInvoice App Store support and privacy pages.

## Pages

- Home: `index.html`
- Support: `support.html`
- Privacy Policy: `privacy.html`

## Before publishing

Replace:
- `YOUR NAME`
- `YOUR-EMAIL@example.com`

## GitHub Pages

Create a public repository named:

`your-github-username.github.io`

Upload these files to the repository root.

Your site will be available at:

`https://your-github-username.github.io/`

App Store URLs:

`https://your-github-username.github.io/support.html`

`https://your-github-username.github.io/privacy.html`
